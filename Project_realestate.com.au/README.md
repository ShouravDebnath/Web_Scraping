# Real Estate Agent Data Scraper (Python)

## Project Overview

This project demonstrates an automated web scraping system built with Python to collect real estate agent data from the website **realestate.com.au**.

The scraper loads dynamic web pages, extracts structured information, and processes the collected data into a clean dataset ready for analysis.

The project showcases skills in **web scraping, browser automation, and data cleaning**.

---

# Project Workflow

Website → Browser Automation → HTML Extraction → Data Parsing → Data Cleaning → Final Dataset
