# Geneon Product Data Scraping & Cleaning Pipeline

## Project Overview

This project demonstrates a complete **web scraping and data processing pipeline** built with Python.  
The goal of the project is to collect product information from the Geneon website, extract structured data from HTML pages, clean the dataset, and produce a final organized dataset suitable for analysis or database storage.

The pipeline includes:

- Automated webpage collection
- HTML parsing
- Data extraction
- Data cleaning
- Structured dataset generation

This project demonstrates real-world **data engineering and data analysis workflows**.

---

# Project Workflow

Website → HTML Scraping → Data Extraction → Data Cleaning → Final Dataset
